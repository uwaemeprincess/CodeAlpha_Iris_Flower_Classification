# CodeAlpha_Iris_Flower_Classification
Iris Flower Classification Project using Python and Scikit-Learn for the CodeAlpha Data Science Internship Task 1

#Project overview
built a machine learning classification model using scikit-learn and
 logistic regression to classify iris flowers into three distinct species(setosa, versicolor, and virginica) based on their structural measurement.

 #performance & metrics
 overall model accuracy:96.67%
 f1-score :0.97
 the model successfully classified 29 out of 30 test samples experiencing a minimal confusion between versicolor and virginica.

 tools used
 python
 pandas
 scikit-learn
 numpy
 joblib(for data serialization)
